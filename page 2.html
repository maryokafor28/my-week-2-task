QUESTIONS AND ANSWERS

HOW DO I COMMIT A FILE TO GITHUB

  1. Select the file you want to upload
  2. Drag and drop the file or folder or click choose your file
  3. in the commit message field, type a short meaningful commit message that describes the change you made to the file
  4. below the commit message, decide whether to add your commit to the current branch or to a new branch

HOW TO FORK A REPOSITORY

  1. On GITHUB, navigate to the octocat/spoon-knife repository
  2. In the top right corner of the page, click fork
  3. under "owner" select the dropdown menu and click an owner for the forked repository
  4. in the description field, tyoe a description of your fork
  5. click create fork
